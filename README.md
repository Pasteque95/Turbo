# Turbo

Turbo is een full-stack webapplicatie ontwikkeld als programming project.

## Beschrijving

Turbo is een digitaal logboek waarmee gebruikers hun oefentrajecten voor het praktijkexamen rijbewijs B kunnen beheren.

## Technologieën

- Next.js
- React
- TypeScript
- Tailwind CSS
- Prisma
- MySQL
- Auth.js
